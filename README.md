## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`NODE_ENV`

`APP_PORT`

`DB_USERNAME`

`DB_PASSWORD`

`DB_NAME`

`DB_HOST`

`DB_PORT`

`JWT_SECRET_KEY`

`JWT_EXPIRES_IN`

`ADMIN_EMAIL`

`ADMIN_PASSWORD`


## Installation

Install my-project with npm

```bash
  npm run install
  npm run migrate
  npm run seed:all
```
